# Балабанов Александр БАС-1 Лабораторная 2
**Задание 1**

**Задача 1**

**Текст задачи**

Дерево содержит строки. Заменить в каждой строке последний символ на заданный.

**Алгоритм решения**

Программа запрашивает у пользователя количество элементов count для построения бинарного дерева. Если count положительное число, программа генерирует count случайных чисел в диапазоне от 1 до 1 000 000, преобразуя их в строки. Из сгенерированного списка строк с помощью рекурсивной функции buildTreeFromList строится бинарное дерево поиска. Функция buildTreeFromList последовательно вставляет каждый элемент списка в дерево, используя функцию insertIntoTree, которая работает следующим образом: если дерево пустое (Empty), создается новый узел Node с переданным значением и пустыми левым и правым поддеревьями; если дерево не пустое, происходит сравнение длин строк — если длина вставляемой строки меньше или равна длине строки в текущем узле, значение вставляется в левое поддерево, иначе — в правое поддерево. Построенное дерево выводится на экран с помощью рекурсивной функции printTree, выполняющей обход в прямом порядке (корень, левое поддерево, правое поддерево) с отступами для отображения структуры. Затем программа запрашивает у пользователя символ symbol для замены и применяет рекурсивную функцию mapStringTree, которая создает новое дерево путем обхода исходного: для каждого узла функция replaceLastChar заменяет последний символ строки на заданный символ (если строка не пустая). Полученное после преобразования дерево выводится на экран с сохранением структуры, но с измененными значениями узлов. Если введенное количество элементов count меньше или равно нулю, программа запрашивает ввод до тех пор, пока не будет введено положительное число, благодаря встроенной проверке в функции readInt. В конце программа возвращает 0.

**Задание 2**

**Задача 1**

**Текст задачи**

Найти сумму четных значений, находящихся в листьях (узел является листом, если у него нет ни левого, ни правого поддерева).

**Алгоритм решения**

Программа запрашивает у пользователя количество элементов count для построения бинарного дерева поиска. Если count положительное число, программа генерирует count случайных чисел в диапазоне от 1 до 100 и выводит их на экран. Из сгенерированного списка чисел с помощью рекурсивной функции buildTreeFromList строится бинарное дерево поиска. Функция buildTreeFromList последовательно вставляет каждый элемент списка в дерево, используя функцию insertIntoTree, которая работает следующим образом: если дерево пустое (Empty), создается новый узел Node с переданным значением и пустыми левым и правым поддеревьями; если дерево не пустое, происходит сравнение значений — если вставляемое значение меньше или равно значению в текущем узле, оно вставляется в левое поддерево, иначе — в правое поддерево. Построенное дерево выводится на экран с помощью рекурсивной функции printTree, выполняющей обход в прямом порядке (корень, левое поддерево, правое поддерево) с отступами для отображения структуры. Затем программа с помощью рекурсивной функции sumEvenInLeaves выполняет подсчет суммы четных значений, находящихся только в листьях дерева. Функция sumEvenInLeaves обходит дерево и для каждого узла проверяет с помощью вспомогательной функции isLeaf, является ли данный узел листом (отсутствуют левый и правый потомки); если узел является листом и его значение четное, оно добавляется к сумме, иначе функция рекурсивно суммирует значения из левого и правого поддеревьев. После завершения подсчета программа выводит полученную сумму на экран. Если введенное количество элементов count меньше или равно нулю, программа запрашивает ввод до тех пор, пока не будет введено положительное число, благодаря встроенной проверке в функции readInt. В конце программа возвращает 0.
