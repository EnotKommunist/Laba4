**Задание 1**

**Задача 1**

**Текст задачи**

Получить список из минимальных цифр натуральных чисел, содержащихся в исходной последовательности

**Алгоритм решения**

Программа запрашивает у пользователя количество элементов count_el. Если count_el больше нуля, программа запрашивает ввод count_el чисел, сохраняя их в список numbers. Для каждого числа из списка вызывается функция findMinNum, которая использует рекурсивную функцию minNum для поиска минимальной цифры. Функция minNum работает так: если число меньше 10, она сравнивает его с текущим минимумом и возвращает меньшее; иначе выделяет последнюю цифру, сравнивает с минимумом и рекурсивно обрабатывает оставшуюся часть числа. Результаты собираются в список minDigits, после чего программа выводит исходные числа и их минимальные цифры. Если count_el меньше или равен нулю, выводится сообщение об ошибке. В конце программа возвращает 0.

**Задание 2**

**Задача 1**

Текст задачи

Найти количество элементов списка, в которых встречается заданная цифра.

**Алгоритм решения**

Программа запрашивает количество элементов countEl. Если countEl больше нуля, она запрашивает ввод countEl чисел, сохраняя их в список numbers, а затем запрашивает искомую цифру requiredNumber. Функция findNum применяет к каждому числу рекурсивную функцию recNum, которая ищет заданную цифру в числе: если число меньше 10, проверяет совпадение; иначе проверяет последнюю цифру и рекурсивно обрабатывает оставшуюся часть. Результаты (1 если цифра найдена, иначе 0) суммируются через Seq.fold, и итог выводится на экран. При некорректном countEl выводится сообщение об ошибке.

**Задание 3**

**Задача 1**

Текст задачи

Вывести самое короткое название файла в указанном каталоге.

**Алгоритм решения**

Программа через рекурсивную функцию getValidDirectory запрашивает путь к существующей директории. После получения пути она получает список файлов через Directory.GetFiles, преобразует его в последовательность и извлекает имена файлов с помощью Path.GetFileName. Если файлов нет, выводится сообщение. Иначе вызывается рекурсивная функция minStringLength, которая обходит последовательность имён: при одном элементе сравнивает его длину с текущим минимумом и возвращает меньшее; при нескольких — берёт первый элемент, обновляет минимум и рекурсивно вызывает себя для оставшихся элементов через Seq.tail. На выходе получается минимальная длина имени файла, которая выводится на экран. Программа завершается возвратом 0.
